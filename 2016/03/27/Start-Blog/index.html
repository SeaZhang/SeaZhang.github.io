前言

平时看博客，发现很多人csdn，cnblog的博客不更新了，都开始自己建个人博客，反正想着最近要沉淀下来，开始把一些东西写下来，不仅仅是保存到云笔记上，于是开始走起来。

最后

本地显示ok后，开始上传到github上去。

hexo安装 hexo-deployer-git

安装成功后进，修改 _config.yml中

deploy:
type: git
repo:
branch: [branch]
message: [message]
